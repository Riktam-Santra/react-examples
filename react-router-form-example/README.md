# React Routing with Form data

## Setup your project
1. Clone your github project into your local computer:
    ```
    git clone https://github.com/Riktam-Santra/react-examples
    ```

2. Open the project in VS Code:
    ```
    code ./react-examples/react-router-form-example
    ```

3. Install any missing packages (DO NOT SKIP):
    ```
    npm i
    ```

4. Run the project

    ```
    npm run dev
    ```
